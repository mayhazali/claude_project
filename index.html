How Claude Was Used

Code Generation and Organization

Claude generated all the HTML, CSS, and JavaScript for this project from scratch. It also refactored the code by extracting inline styles into a shared styles.css file, keeping both pages consistent and easier to maintain.

Launch on Localhost

Claude provided the command to serve the site locally using Python's built-in HTTP server (python3 -m http.server 8000) and diagnosed why the directory listing was appearing instead of the page — the server was being run from the wrong directory.

APOD API

Claude built the NASA Astronomy Picture of the Day page, including fetching data from NASA's public API, handling both image and video media types, and displaying the title, date, and description dynamically.

HTML Responsive Web Design

Claude audited the site for mobile compatibility, identified issues with fixed video height, oversized fonts, and tight padding on small screens, then added CSS media queries to make the site look good on phones and all screen sizes.

Connection to GitHub and GitHub Pages

Claude checked whether Git and the GitHub CLI were installed, walked through initializing a git repository, connecting it to a remote GitHub repo, and pushing the code. It also provided instructions to enable GitHub Pages for free hosting.

Estimated Time Saved

Building this project manually — writing the HTML, CSS, and JavaScript, debugging the localhost issue, integrating the NASA API, making the site responsive, and setting up GitHub — would typically take a developer 3–5 hours. With Claude, it was done in a single conversation in under 30 minutes, saving an estimated 3–4.5 hours of work.
